Git Diff介绍

Feb 2nd, 2015 12:14 am

在CVS系统中，我们经常使用diff命令查看文件的修改记录。本章将介绍在Git中怎样进行diff，以及Git diff的一些特殊使用方法。

Git diff命令的几种形式

  • git diff

    比较工作路径和索引中文件的区别。他将显示工作路径下需要添加至索引中的内容。

  • git diff commit

    比较工作路径和指定提交之间的区别。

  • git diff –cached commit

    比较添加至索引的内容和指定提交之间的区别。

  • git diff commit commit

    比较任意指定的两个提交之间的区别。
